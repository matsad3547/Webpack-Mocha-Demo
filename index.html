How to Set Up Webpack

  1. Create directory structure
      2.       --parent dir--
        -/src/
           -/styles/
           -/scripts/
           -/tests/
        -/public/
  2. $ npm init
  3. $ npm install webpack --save-dev
  4. Create webpack.config.js file at parent level and populate it
  5. Add a entry.js file under /scripts/ and put the required file names in there
  6. Install style, sass, and css loaders: $ npm install style-loader css-loader sass-loader node-sass --save-dev
  7. Run webpack: $ webpack --watch
  8. Add webpack server: $ npm install webpack-dev-server --save-dev
  9. Open a new terminal window and run the webpack server in that window: $ node_modules/.bin/webpack-dev-server
  10. NOTE: Both terminal windows need to be open, running both webpack --watch AND webpack-dev-server
  11. In the browser, navigate to: localhost:8080/webpack-dev-server/

How to Set Up Mocha for TDD

  1. Install Mocha in the parent directory: $ npm install mocha --save-dev
  2. Add a 'test' folder
  3. Add a tests.js file - as additional features are created, add a file for those tests as feature.tests.js
  4. Add tests to the tests.js
  5. tests have the following format: 
    var assert = require('assert');

      describe('Array', function () {
        describe('indexOf(#)', function() {
          it ('should return -1 when the value is not present', function () {
            assert.equal(-1, [1, 2, 3].indexOf(4));
          });
        });
      });
  6. Run the tests: $ mocha ./src/test
  7. Add to the package.json: "test": "mocha ./src/test/"
  8. Then you can run all the tests with $ npm test
  9. You can run tests individually or in groups with : $ mocha ./src/test specifying the directory and/or filename or use the ones from here: npm-scripts

How to Setup a React Project with Sass

  1. Setup an Initial File Structure
      2.         --parent dir--
          -/src/
             -/app/
             -/tests/
  2. Create the .gitignore file. Populate it using the tags Sass, OSX, macOS, Node from here
  3. Create a index.html file in /src/
  4. Run $ npm init
  5. Fill in the questions as you please
  6. Install webpack: $ npm install webpack --save-dev
  7. Edit the package.json file to include all the required dependencies:
      8.           {
            "name": "Project",
            "version": "1.0.0",
            "description": "Project Description",
            "main": "index.js",
            "scripts": {
              "build": "webpack -d && cp src/index.html dist/index.html && webpack-dev-server --content-base src/ --inline --hot",
              "start": "npm run build",
              "build:prod": "webpack -p && cp src/index.html dist/index.html"
            },
            "repository": {
              "type": "git",
              "url": "Project"
            },
            "author": "You",
            "license": "ISC",
            "devDependencies": {
              "babel-core": "^6.18.0",
              "babel-loader": "^6.2.7",
              "babel-preset-es2015": "^6.18.0",
              "babel-preset-react": "^6.16.0",
              "node-sass": "^3.10.1",
              "react-hot-loader": "^3.0.0-beta.6",
              "sass-loader": "^4.0.2",
              "webpack": "^1.13.3",
              "webpack-dev-server": "^1.16.2"
            },
            "dependencies": {
              "react": "^15.3.2",
              "react-dom": "^15.3.2"
            }
          }
  8. Run npm: $ npm start
